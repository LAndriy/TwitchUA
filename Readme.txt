# ТвічЮА - Українська локалізація для Twitch.tv

Розширення для браузера, яке додає українську локалізацію до платформи Twitch.tv.

## Про розширення
ТвічЮА — це браузерне розширення, яке автоматично перекладає інтерфейс Twitch.tv українською мовою.
На жаль, Twitch не надає інструментів для роботи з текстом, тому розширення шукає конкретний текст і заміняє його на українську версію.
Тому деякі інтерактивні елементи, що містять посилання,імена стримерів та інші елементи, не будуть перекладені через особливості роботи скрипту.
Розширення базується на відкритому коді та базі перекладів версії 2.3, створеної командою перекладачів:

- AgeNTsGame (https://www.twitch.tv/agentsgame)
- TerryHunteR (https://www.twitch.tv/terryhunter)
- smile_hamster (https://www.twitch.tv/smile_hamster)
- duckmale (https://www.twitch.tv/duckmale)

Особлива подяка:
- oldboiua (https://www.twitch.tv/oldboiua)

Цю версіюможна встановити перейшовиши з посиланням - https://chrome-stats.com/d/abgcdoojabobifkadelcggbfejppmhob

### Версія 2.4 (поточна)
- Підвищена швидкість роботи
- Додано нові елементи, які підлягають перекладу

- Ведеться постійна робота над перевіркою та покращенням якості перекладів

Щоб встановити розширення вам потрібно перейти в режим розробника у вашому браузері, для цього виконайте такі кроки:

1.Відкрийте Chrome і перейдіть на сторінку розширень:

2.В адресному рядку введіть chrome://extensions/ або about:addons і натисніть Enter.

Увімкніть режим розробника:
3.У верхньому правому куті сторінки розширень знайдіть перемикач Режим розробника (Developer mode) і увімкніть його.

4.Завантажте розширення:
Тепер ви побачите декілька опцій для завантаження розширень:
Вам потрібно: "Завантажити розпаковане розширення(Load unpacked)"
Виберіть папку: відкриється вікно, де потрібно вибрати папку з кодом розширення (це має бути папка з усіма файлами розширення, а не сам архів).

5.Розширення активується автоматично, якщо ні тоді активуйте розширення:
Ви побачите його на сторінці chrome://extensions/ та можете ввімкнути/вимкнути його за допомогою перемикача.
